# Проектная работа Mesto
студент Власенко Елена

**Mesto** - это интерактивная страница, где пользователи могут размещать фотографии, редактировать свой профиль и взаимодействовать с контентом других пользователей. Проект реализован как SPA (Single Page Application).

Проект развернут и доступен по адресу: [GitHub Pages](https://vlasenkoElena-mag.github.io/mesto-project-ff/)

##  Функциональность

### Пользовательский профиль
- **Редактирование информации** - изменение имени и описания профиля
- **Обновление аватара** - загрузка новой фотографии профиля
- **Отображение данных** - имя, описание и аватар пользователя

### Работа с карточками
- **Просмотр фотографий** - открытие изображений в полном размере
- **Добавление новых карточек** - создание постов с названием и изображением
- **Удаление собственных карточек** - управление своим контентом
- **Система лайков** - возможность ставить/убирать лайки карточкам

### Пользовательский интерфейс
- **Адаптивный дизайн** - корректное отображение на всех устройствах
- **Модальные окна** - для редактирования профиля, добавления карточек
- **Валидация форм** - проверка корректности введенных данных
- **Состояния загрузки** - индикация процесса сохранения/удаления

## Технологии

### Основной стек
- **JavaScript (ES6+)** - современный JavaScript с использованием модулей
- **CSS3** - стилизация с BEM методологией
- **HTML5** - семантическая разметка

### Инструменты сборки и разработки
- **Webpack 5.101.3** - модульный сборщик проекта
- **Babel 7.28.4** - компилятор JavaScript для поддержки старых браузеров
- **PostCSS** - обработка CSS с автопрефиксером

## Установка и запуск

### Предварительные требования
- Node.js (версия 14 и выше)
- npm или yarn

## Клонирование репозитория
git clone https://github.com/vlasenkoElena-mag/mesto-project-ff.git
cd mesto-project-ff

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка проекта
npm run build

# Деплой на GitHub Pages
npm run deploy
